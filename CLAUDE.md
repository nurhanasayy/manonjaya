# Manon Jaya

Landing page for Manon Jaya - a construction materials store (Toko Bahan Bangunan).

## Tech Stack

- **Framework**: TanStack Start 1.132.0 (React 19)
- **Routing**: TanStack Router (file-based)
- **Styling**: Tailwind CSS 4
- **Build**: Vite 7
- **Language**: TypeScript
- **Linting**: Biome

## Project Structure

```
src/
├── routes/
│   ├── __root.tsx    # Root layout with head/meta
│   └── index.tsx     # Home page (landing page)
├── router.tsx        # Router configuration
└── styles.css        # Global styles (Tailwind)
```

## Development

```bash
pnpm dev      # Start dev server on port 3000
pnpm build    # Production build
pnpm test     # Run tests
pnpm check    # Lint and format check
```

## Landing Page Requirements

**Business**: Construction materials store
**Target**: Customers looking for building supplies
